# ⚙️ wazOS v66.0 (Biomechanical Logic Engine)

wazOS ist ein modulares Framework für mobile und hybride Generatoren auf Plattformen wie Perchance. Es fokussiert sich auf präzises Rigging, anatomische Auto-Balancierung und mobile UX (CRT-Terminal, X-Ray Touch-Offsets).

## 🚀 Kern-Features
* **Modular Architecture:** Plugins für Biped, Hand, Face & Multi-Limb Rigs (Goro, Hydra).
* **Anatomy Shield:** Biologische Limits und Auto-Balancing für Keyframe-Interpolation.
* **Mobile First UX:** Emoji-Shape Hitboxes, X-Ray Geister-Punkte und Auto-Opacity.
* **Self-Healing Core:** 10-Step History Buffer & Live Logcat-Terminal.

## 📁 Struktur
* `/src/core` - System Manager, History und Guard Module.
* `/src/biomechanics` - Gelenk-Limits und Symmetrie-Engines.
* `/src/ui` - Das reaktive SVG-Interface und die CRT-Styles.

## 🛠 Entwicklung
Dieses Projekt nutzt isolierte JS-Module, die für den Einsatz in Text-to-Image Generatoren (z.B. Perchance Vortex) konzipiert sind.
